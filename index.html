¿Qué es el internet?

Internet es una red global de equipos informáticos que se comunican mediante un lenguaje común. Es decir, es la red de redes, miles de ordenadores por todo el mundo conectados entre sí. Al conectarse a este sitio Web se ha conectado a Internet. Es similar al sistema de teléfonos internacional: nadie posee ni controla todo el conjunto, pero está conectado de forma que funcione como una gran red.
